# 💰 스마트 개인금융관리 시스템

## 📌 프로젝트 소개

스마트 개인금융관리 시스템은 사용자의 수입과 지출을 효율적으로 관리하고,
금융 데이터를 기반으로 소비 패턴을 분석하여 합리적인 소비 습관을 형성할 수 있도록 돕는 웹 서비스입니다.

---

## 🎯 개발 목적

* 개인의 금융 데이터를 체계적으로 관리
* 소비 패턴 분석을 통한 지출 최적화
* 직관적인 UI를 통해 누구나 쉽게 사용 가능

---

## 🛠️ 사용 기술

### Backend

* Java (Spring Boot)
* MyBatis
* Oracle DB

### Frontend

* HTML / CSS / JavaScript
* JSP

### 기타

* Git / GitHub
* Docker (배포 예정)

---

## 📂 주요 기능

### 👤 회원 관리

* 회원가입 / 로그인
* 사용자 정보 조회 및 수정

### 💸 수입/지출 관리

* 수입 및 지출 등록
* 카테고리별 관리
* 월별/일별 조회

### 📊 통계 및 분석

* 소비 패턴 분석
* 카테고리별 지출 비율
* 그래프 기반 시각화

### 🔔 알림 기능 (예정)

* 예산 초과 시 알림
* 정기 지출 알림
